Mission:
We exist to make the web navigable and usable for neurodivergent individuals, and to create a data flywheel that helps businesses understand and design for neurodivergent interaction patterns.

What We Do:
We build assistive technology that reduces cognitive friction for neurodivergent users on real websites, while generating scalable behavioural insights that help businesses design better digital experiences.

Core Product:

B2B2C SaaS Marketplace – Latch:
• User testing by NDs at scale, via optimised focus group matching
• Immediate and regular actionable recommendations from a powerful specialised dataset
• Community & easy apply jobs for ND testers

Chrome extension – Neuro Assist:
• Data flywheel engine
• Captures user intent before they interact with a website
• Assists users in completing tasks with clarity and reduced cognitive load
• Measures whether the website successfully supported that intent
• Aggregates anonymised data to reveal where websites fail neurodivergent users
